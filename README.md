# AI Stock Analyzer

An AI-powered stock analysis tool that provides technical indicator analysis with buy/sell/hold signals. Supports BIST, US, European stocks and crypto. Analysis is split by investment horizon (short, medium, long term).

## Live Demo

[Try it here](https://bunyaminkeremon.github.io/ai-stock-analyzer/)

## What It Does

- Analyze any stock by symbol (BIST, US, Europe, Crypto)
- Quick select popular stocks (THYAO, GARAN, ASELS, AAPL, TSLA, BTC)
- Technical indicators: RSI, MACD, Bollinger Bands, SMA, EMA, ADX, Stochastic RSI, Volume
- Buy/Sell/Hold signal with confidence percentage
- Separate analysis for short term (1-7 days), medium term (1-3 months), long term (6+ months)
- Support and resistance levels, stop loss, target prices
- Risk tolerance setting (low, medium, high)

## Tech Stack

- HTML5
- CSS3
- JavaScript (async/await, Fetch API)
- Google Gemini API for AI analysis

## Run Locally
```bash
git clone https://github.com/bunyaminkeremon/ai-stock-analyzer.git
```

Open `index.html` in your browser. You'll need your own API key from aistudio.google.com — replace the key in `script.js`.
